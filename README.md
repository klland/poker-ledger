# 牌桌帳房

手機優先的德撲籌碼紀錄 PWA，支援：

- 花順俱樂部：1 籌碼 = NT$1
- 馬來俱樂部：1 籌碼 = NT$8
- 兩館合計：各自換算台幣後合併顯示總價值與總輸贏
- 買入／加碼、領出與每日結算
- 每筆紀錄皆可調整日期與時間，支援補登遺漏資料並自動重算後續損益
- 完整紀錄可逐筆刪除，刪除後自動重算餘額與統計
- 日結算紀錄、本週／本月／全部統計
- 淨輸贏、勝率、最佳單日、平均單日與走勢圖
- 離線使用

## 結算規則

每天只需輸入「今天玩完後手上剩餘的全部籌碼」，不需自行輸入單日獲利。

本次輸贏籌碼 = 今天剩餘總籌碼 + 本次結算前領出 − 上次剩餘總籌碼 − 本次結算前新增

因此新增籌碼不算獲利；領出籌碼也不會改變累計淨輸贏。未領出的籌碼會保留，直接延續到下一次使用。

## 本機開啟

在此資料夾執行：

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

再開啟 `http://127.0.0.1:4173`。部署到 HTTPS 網址後，可用 Safari 或 Chrome 的「加入主畫面」安裝成 App。

資料保存在目前瀏覽器的本機儲存空間；清除瀏覽器資料時，紀錄也會一併刪除。

## 驗證

`tests/browser-qa.cjs` 會驗證馬來俱樂部的核心流程：買入 10,000 籌碼、收桌 12,000 籌碼應為獲利 NT$16,000，領出 2,000 籌碼後累計淨輸贏仍維持 NT$16,000。
